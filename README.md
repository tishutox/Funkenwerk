# Funkenwerk Webseite

Dieses Repository enthält eine überarbeitete Webseite für das Funkenwerk der Technischen Hochschule Augsburg.

Im Rahmen einer Aufgabe an der TH Augsburg im Modul **"Grundlagen betrieblicher Informationssysteme"** sollten wir eine Webseite unserer Wahl auswählen und gestalterisch sowie strukturell verbessern.

Wir haben uns für die Funkenwerk-Seite entschieden und dabei unter anderem folgende Punkte umgesetzt:

- klarere Seitenstruktur und bessere Lesbarkeit
- moderne, responsive Darstellung für Desktop und Mobile
- aufgeräumte Navigation mit Dropdowns
- inhaltliche Überarbeitung einzelner Seiten

Ziel war es, die Inhalte ansprechender zu präsentieren und die Benutzerfreundlichkeit der Seite zu verbessern.
